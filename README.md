# YouTube Downloader

## [https://you-tube-downloader-six.vercel.app/](https://you-tube-downloader-six.vercel.app/)

## Опис


YouTube Downloader - це веб-застосунок, що дозволяє користувачам завантажувати відео з YouTube, ввівши посилання на відео. Проєкт створено з навчальною метою: опанування роботи з API,станом UI та обробкою асинхронних запитів.

## Стек

- React 18
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Shadcn/ui

## Початок роботи

```bash
git clone https://github.com/OleksiyNikolaenko/YouTube-Downloader.git
cd youtube-downloader
npm install
npm run dev
```
